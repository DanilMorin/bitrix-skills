# Подготовь отчёт для будущей посадки на Bitrix

Составь отчёт, который можно будет передать Bitrix-разработчику.

В отчёте должны быть:

1. Карта шаблона сайта.
2. Карта компонентов.
3. Карта инфоблоков.
4. Список include areas.
5. Список форм.
6. Список JS-логики.
7. Риски.
8. Что уже подготовлено.
9. Что нужно сделать при посадке.
