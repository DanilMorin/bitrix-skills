# Финальный чеклист Bitrix-ready аудита

## Перед изменениями

- Понять структуру проекта.
- Найти package.json.
- Понять сборку.
- Понять, где HTML, SCSS, JS.
- Проверить, есть ли GitHub Pages preview.

## Во время аудита

- Найти повторяемые блоки.
- Найти будущие компоненты.
- Найти будущие include areas.
- Найти будущие инфоблоки.
- Найти проблемы с HTML.
- Найти проблемы с SCSS.
- Найти проблемы с JS.

## После изменений

- Проверить сборку.
- Проверить адаптивность.
- Проверить визуал.
- Проверить консоль браузера.
- Проверить, что GitHub Pages не сломан.
- Составить отчёт.
