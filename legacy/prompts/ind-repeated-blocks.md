# Найди повторяющиеся блоки

Проанализируй HTML-файлы проекта.

Найди блоки, у которых:

- одинаковая или похожая структура;
- повторяются карточки;
- меняется только контент;
- похожие классы;
- одинаковое JS-поведение.

Для каждого блока укажи:

- где найден;
- что повторяется;
- что меняется;
- можно ли объединить;
- будущую Bitrix-сущность;
- безопасное действие сейчас;
- риск.
